# Season 0: The Archive (winner only)

The weeks we played before full tracking began. We have no detailed stats for these,
so this season records only who won each Monday. Backfill from memory whenever the old
winners come back to you by adding match objects to the `matches` array below.

A match needs only a date and a winner. `winner` can be a single slug, or an array of
slugs for a tie. `note` is optional.

```json
{
  "name": "Super Battle Golf League",
  "season": "S0",
  "seasonLabel": "Season 0: The Archive",
  "updated": "Jun 10, 2026 · midday PT",
  "matches": [
    { "date": "May 4",  "winner": "cunder" },
    { "date": "May 11", "winner": "cunder", "note": "Back to back." },
    { "date": "May 18", "winner": "toiletduck" },
    { "date": "May 25", "winner": "cunder", "note": "Three of five. The preseason was the Cunder show." },
    { "date": "Jun 1",  "winner": "sandman" }
  ]
}
```
